PDF Accessibility Engine

v0.1.0

A local-first pipeline that adds PDF/UA accessibility tags, validates structure, repairs corruption, audits font health, and redacts sensitive data — all while preserving the original native PDF content. No cloud upload required.

Capabilities

  • A Accessibility Tagging
  • N Native PDF Preservation
  • F Font Health Audit
  • R Corruption Repair
  • S SSN Redaction
  • V PDF/UA Validation

Native PDF Retained

🛡️ NATIVE PDF RETAINED

When this badge appears, the original vector text, fonts, links, and form fields have been preserved exactly. No rasterization. Text stays sharp at any zoom level, and screen readers can access the real content.
